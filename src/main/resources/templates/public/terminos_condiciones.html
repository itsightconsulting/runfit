1.1. Programa Personalizado de Entrenamiento.

El PROGRAMA DE ENTRENAMIENTO, será diseñado por Runfit de forma personalizada para el CONTRATANTE con la siguiente información:

Éstas se realizarán dos veces al mes y tienen un tiempo de duración de 90 minutos cada una. Las sesiones no son de carácter obligatorio por parte de Runfit, son un servicio opcional para reforzar el proceso de asesoría.
